# Expression Calculator using C

This project evaluates mathematical expressions using stack.

## Features
- Supports +, -, *, /
- Handles brackets
- Supports multi-digit numbers

## Example
Input: (2+3)*4  
Output: 20

## What I learned
- Stack implementation
- Expression evaluation logic
- Operator precedence
